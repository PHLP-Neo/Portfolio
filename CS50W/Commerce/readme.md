# CS50 Web - Commerce Auction Site

**eBay-like auction platform** in Django: listings/bids/comments/watchlists/categories/admin.

## Features Implemented
- **Listings**: Create active listings (title/desc/image/category/start bid).
- **Index/Listing Pages**: All active + details (current bid, watchlist toggle, bid form > current).
- **Actions**: Close auction (winner), add/remove comments/watchlist.
- **Extras**: Categories filter, Django admin CRUD.
- Tech: Custom models (Listing/Bid/Comment), forms/validation.[file:79]

**What I Learned**: Model relationships (FK/M2M), bid logic/security.
