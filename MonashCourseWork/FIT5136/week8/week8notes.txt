Example 2: Gym Eligibility

Gym Eligibility Specification: 
	Return True if a person age (integer) is between 18-65, or False otherwise. 

Possible Categories: Age < 0, 0 <= Age < 18, 18 <= Age <= 65, Age > 65 

Test Scenarios/Test Cases: ----
	Negative Scenario: Age(-5) = False (Invalid, age cannot be negative) 
	Positive Scenario: Age(10) = False (Valid, falls in the range 0-17) 
	Positive Scenario: Age(25) = True (Valid, falls in the range 18-65) 
	Positive Scenario: Age(75) = False (Valid, falls in the range 65+)
