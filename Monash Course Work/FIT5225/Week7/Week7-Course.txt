40. Boot Strapping:
When you launch an AWS resource, you will have an instance with a default configuration. 
Then, you have to install software and do custom configuration to bring it to a certain state.
Bootstrapping enables you to script software installation and configuration and execute it while launching the instance.
Instance Metadata and User Data (Information about an instance)
	curl http://169.254.169.254/latest/user-data/ -> your bootstrapping data
	curl http://169.254.169.254/latest/meta-data/ -> examples: public-ipv4, hostname and etc.
The IP address 169.254.169.254 is a link-local address and is valid only from the instance.
